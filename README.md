  
# Mémoire – Hilaire Barbot

## 📘 Présentation

Ce dépôt contient mon mémoire universitaire portant sur l’étude des câbles sous-marins de télécommunication.

Les câbles sous-marins sont des infrastructures stratégiques devenues le système nerveux de l’économie mondiale.

Ce travail vise à expliquer leur fonctionnement, à la fois sur le plan économique et technique.

## 📚 Contenu du projet

- Document du mémoire au format PDF  
- Travaux d’analyse sur les câbles sous-marins  
- Ressources utilisées pour la recherche  

## 🔎 Objectif du travail

Après avoir présenté l’histoire des câbles sous-marins, leurs origines et leur fonctionnement, ce mémoire examine les principaux acteurs du milieu, ainsi que les enjeux et les perspectives d’évolution.
